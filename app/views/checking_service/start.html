Check your ERS files

Use this service to check for formatting errors in the spreadsheet or comma separated values (CSV, .csv) files that you need to upload as part of your Employment Related Securities (ERS) annual return.

Before you start

You'll need to know which type of ERS scheme or arrangement the file is for.

If you are checking a spreadsheet it needs to be an OpenDocument Spreadsheet (.ods file).

The time it takes to check a file depends on the speed of your internet connection and the size of your spreadsheet or CSV file.
