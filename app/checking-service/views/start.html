Check your ERS files

Use this service to check for formatting errors in the files you need to upload as part of your Employment Related Securities (ERS) annual return.

Before you start

You'll need to know which type of ERS scheme or arrangement the file is for.

If you are checking a spreadsheet it needs to be an OpenDocument Spreadsheet (.ods) file. You can also check comma separated values (CSV, .csv) files.

The time it takes to check a file depends on the speed of your internet connection and the size of your spreadsheet or CSV file.

Use the ERS guidance documents to help you create error-free files.
